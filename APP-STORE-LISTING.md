# TABWISE — Smart Tab Optimizer

## Short Description
Take control of tab chaos with smart grouping, RAM-saving suspension, and focus-first tab management.

## Full Description
TABWISE is a Chrome extension built for people who keep lots of tabs open and want a faster, cleaner browser workflow.

It combines tab organization, duplicate cleanup, read-later workflows, and weekly productivity insights in one compact popup.

### What TABWISE does
- Groups tabs by domain so you can instantly see where your browser attention is going
- Detects duplicate tabs and lets you close extras quickly
- Helps reduce tab overload with Dust indicators (tab age) and Budget tab limits
- Supports one-click tab suspension and restore for memory relief
- Lets you soft-close tabs into a Read Later queue and reopen when ready
- Generates a local weekly report with revisit behavior and top domains

### Key Features
- **Tabs Dashboard**: tab count, estimated RAM, stale-tab indicator, duplicate domain count
- **Domain Navigation**: click rows to jump directly to tabs
- **Duplicate Controls**: expand duplicate groups and close extra copies
- **Smart Clusters**: auto-group tabs by intent (Work, Research, Shopping, Entertainment)
- **Cluster Actions**: close all tabs in a cluster or save a session snapshot
- **Budget (Pro)**: set a max active-tab budget with visual pressure indicators
- **Read Later Queue**: save and close tabs now, reopen later
- **Weekly Report (Pro)**: opened tabs, revisit rate, estimated RAM freed, top domains

### Privacy
- TABWISE operates locally in your browser
- Data is stored using `chrome.storage.local`
- No external server is required for core tab management

### Notes
- RAM values are estimated using tab type and activity signals (not OS-level per-tab telemetry)
- Pro features are gated in-app and prepared for payment integration

## Feature Tiers (Current)
